Noteful
=======

Create and save notes with noteful!

Getting Started
=======
1. clone the repo
2. bundle install
3. rake db:setup   # will create a seed user: admin@admin.com, pass:admin123
4. rails server    # start the server 

Running Tests
=======
rspec or rake spec
